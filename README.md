# 🌊 Ocean Depths — Interactive Storytelling Experience

## 🚀 Overview

**Ocean Depths** is an immersive, scroll-driven web experience that takes users on a cinematic journey from the ocean surface to the deepest trenches on Earth.

Instead of a traditional static webpage, this project transforms learning into an **interactive exploration**, where users dive through different ocean layers and discover how life, light, and pressure change with depth.

The experience combines **storytelling, animation, and interaction design** to simulate a real underwater descent.

---

## 🎯 Concept & Inspiration

This project is inspired by:

* 🌐 Awwwards-winning interactive websites
* 🍎 Apple-style storytelling experiences
* 🌊 National Geographic deep-sea documentaries

The goal was to create a **visual + educational journey**, not just a webpage.

---

## 🧩 Features

### 🌊 1. Story-Driven Structure

The website is divided into **5 immersive sections**:

1. **Surface** — Bright sky, waves, and ocean entry
2. **Sunlight Zone (0–200m)** — Rich marine life and photosynthesis
3. **Twilight Zone (200–1000m)** — Fading light and glowing creatures
4. **Midnight Zone (1000–4000m)** — Darkness and extreme pressure
5. **Abyss & Hadal Zone (4000m+)** — Deepest unexplored regions

---

### 🎮 2. Interactive Experience

* 🖱️ Hover animations on creature cards
* 📜 Scroll-based depth simulation
* 🐟 Animated marine elements (fish, jellyfish)
* 💡 Visual transitions based on depth

---

### 🎞️ 3. Smooth Animations

* GSAP-powered scroll animations
* Section reveal effects
* Floating and movement animations
* Cinematic transitions between layers

---

### 🎨 4. Visual Design

* Realistic ocean-inspired color gradients
* Glassmorphism UI cards
* Clean typography (Playfair Display + Inter)
* Depth-based visual storytelling

---

### 📊 5. Real-Time Depth Indicator

* Displays simulated depth based on scroll
* Enhances realism of underwater journey

---
